对数器
  思路：
       有一个你待测的方法a，以及一个不计时间成本和空间成本但稳定的方法b
       对同一数组（不是同一个地址的数组，是指数组长度、内容值相同的）分别使用两个方法运行
       得到的两个结果进行一一比对，可以验证待测方法的运行结果是否正确
  
  优点：
       不用依赖平台OG来验证自己写的方法是否正确
       比对次数可以自己定义
       每次比对的数组长度、数组内容可以自己定义范围
       待发现错误时，可以通过缩小数组长度、数值范围等控制变量来进行debug修正
